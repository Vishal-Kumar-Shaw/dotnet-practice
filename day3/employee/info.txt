2️⃣ Auto-property kya hota hai?

1. Field means only variables of the class like name, age
property means values along with getter and setter in which we use shortcut like public int age {get; set;}, it replaces
getter and setter method actual implementation
2. Auto-property means when we dont need any validation
3. This real use comes when parameter name and property name is same, this is used to signify object. e.g this.age = age;


⭐ Perfect Answer (Polished Version)

Field = raw data variable inside class
Property = data + controlled access (getter/setter logic)

Property → encapsulation enforce karti hai.
Field → direct memory storage hota hai.
