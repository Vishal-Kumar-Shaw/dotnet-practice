‚≠ê PERFECT INTERVIEW ANSWER

Explicit interface implementation tab use hota hai jab:

‚úî 1. Two interfaces me same method name ho

Example:

interface IB { void Print(); }


Agar dono ko implement karna ho:

{
    void IA.Print() { Console.WriteLine("IA Print"); }
    void IB.Print() { Console.WriteLine("IB Print"); }
}


Yeh ambiguity solve karta hai.

‚úî 2. Method ko public nahi karna chahte

MyLogger obj = new MyLogger();

Yani method hide ho jaata hai.
