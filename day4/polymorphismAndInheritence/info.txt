‚≠ê Correct Interview Answer: override vs new
‚úî override keyword

Override parent ke virtual method ko modify karta hai.

Runtime polymorphism enable karta hai.

Call object ke type se decide hota hai.

public override void Speak() { }

‚úî new keyword (in inheritance)

Parent method ko hide karta hai, override nahi karta.

Runtime polymorphism nahi hota.

Call reference type ke hisaab se hota hai, object type ke hisaab se nahi.

public new void Speak() { }

üß† Short Example
class A
{
    public void Show() { Console.WriteLine("A"); }
}

class B : A
{
    public new void Show() { Console.WriteLine("B"); }
}

A obj = new B();
obj.Show();  // Output: A (because new hides, does not override)
